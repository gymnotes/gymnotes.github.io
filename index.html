Gym Notes

Workouts in your pocket

Description

  • Create your own workout routines
  • iOS 7 User Interface
  • Timer to see how long you have been training. Avoids resting too much!

Note

This app is intended for people who already know exercise names and how to do it. This app does not teach you how to do exercises.
